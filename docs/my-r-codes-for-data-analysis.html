Chapter 3 My R Codes For Data Analysis

In this repository I am going to collect R codes for data analysis.

The title says “My R Codes” but I am only the collector. I will try to refer the original sources as far as I can. Serdar Balci, MD, Pathologist

My aim is to collect all the codes one needs, where one starts with an excel or spss file and then end with the most common analysis used in histopathology papers : example table

There are plenty of ways to do an analysis in R, which is great but also confusing for the newbies. I will collect the codes here so that I can refer later and then update them as I learn more.
